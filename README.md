# 🏨 HotelReservation

A hotel reservation management system built with **ASP.NET MVC** and **SQL Server**, covering the full guest lifecycle from booking to check-out.

---

## Features

- 🔐 **Authentication** — User registration, login, and role-based access (Admin / Guest)
- 🛠️ **Admin Panel** — Manage rooms, reservations, and user accounts
- 🛏️ **Room Management** — Track availability, room types, and pricing
- 📅 **Reservations** — Create, edit, and cancel bookings with conflict validation

---

## Tech Stack

| | |
|---|---|
| Backend | ASP.NET MVC (.NET) |
| Frontend | Razor, HTML5, CSS3, JavaScript |
| Database | SQL Server (MSSQL) + Entity Framework Core |

---

## Getting Started

```bash
git clone https://github.com/ilayybaran/HotelReservation.git
```

1. Update the connection string in `appsettings.json`
2. Run `dotnet ef database update`
3. Start with `dotnet run` or press `F5` in Visual Studio
